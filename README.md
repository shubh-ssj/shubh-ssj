## 📂 Projects

### 🎬 [FFmpex](https://github.com/shubh-ssj/ffmpex) &nbsp;—&nbsp; FFmpeg GUI Wrapper

> A clean, dark-themed GUI wrapper for FFmpeg built with CustomTkinter.
> Convert, compress, trim, and batch-process video & audio without touching the command line.

![Python](https://img.shields.io/badge/Python-ff6b1a?style=flat-square&logo=python&logoColor=white)
![CustomTkinter](https://img.shields.io/badge/CustomTkinter-ff6b1a?style=flat-square&logoColor=white)
![FFmpeg](https://img.shields.io/badge/FFmpeg-ff6b1a?style=flat-square&logoColor=white)
![Version](https://img.shields.io/badge/version-v2.4-ff6b1a?style=flat-square)
[![Stars](https://img.shields.io/github/stars/shubh-ssj/ffmpex?style=flat-square&color=ff6b1a&logo=github)](https://github.com/shubh-ssj/ffmpex/stargazers)

**Key features:** 16+ video formats · 13+ audio codecs · 30+ platform presets · NVENC / AMF / QSV hardware encoding · Job queue · Drag & drop
